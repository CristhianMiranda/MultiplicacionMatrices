Multiplicación de matrices
Esta aplicación realiza la multiplicación de dos matrices cuadradas de tamaño n x n. Además, se proporciona una herramienta para evaluar el rendimiento de diferentes algoritmos de multiplicación de matrices.

Funcionamiento
Existen dos formas de utilizar la aplicación:

1. Ejecución sencilla
    a. Diríjase a la carpeta raíz del proyecto y ejecute el archivo "01 - Ejecutable".
    b. La aplicación generará dos gráficos de barras en la carpeta "assets":
        - Un gráfico muestra la media del tiempo de ejecución de cada algoritmo durante todos los casos.
        - El otro gráfico muestra qué algoritmo es más eficiente para una matriz de tamaño n*n = 1440*1440.
        - La aplicación también genera un archivo de Excel llamado "datos.xlsx" en la carpeta raíz del proyecto. Este archivo contiene la media, varianza, desviación y rango   para cada algoritmo, así como gráficos que muestran el rendimiento de cada algoritmo.
        -Los datos también se guardan en archivos de texto en la carpeta "assets".

2. Ejecución en IntelliJ IDEA
    a. Abra el proyecto en IntelliJ IDEA.
    b. Ejecute la tarea "MultiplicacionMatrices[runStatsTable]".
        - La aplicación mostrará una ventana para cada algoritmo de multiplicación de matrices, donde se mostrará la media, varianza, desviación y rango para cada algoritmo.
        Además, se generarán dos gráficos de barras en la carpeta "assets":
        - Un gráfico muestra la media del tiempo de ejecución de cada algoritmo durante todos los casos.
        - El otro gráfico muestra qué algoritmo es más eficiente para una matriz de tamaño n*n = 1440*1440.
        - La aplicación también genera un archivo de Excel llamado "datos.xlsx" en la carpeta raíz del proyecto. Este archivo contiene la media, varianza, desviación y rango para cada algoritmo, así como gráficos que muestran el rendimiento de cada algoritmo.
        - Los datos también se guardan en archivos de texto en la carpeta "assets".
